# วิธีติดตั้ง Google AdSense บนเว็บไซต์เกมนี้

คู่มือนี้จะสอนวิธีเอา **ID** (รหัสประจำตัว) จาก Google AdSense มาใส่ในโค้ดเพื่อให้โฆษณาแสดงผลครับ ไม่ต้องเขียนโค้ดเพิ่ม แค่ก็อปปี้วางตามจุดที่บอก!

## ขั้นตอนที่ 1: เตรียม Client ID (Publisher ID)
นี่คือรหัสประจำตัวของคุณกับ Google AdSense ใช้เพื่อบอกว่าเว็บนี้เป็นของคุณ

1.  ไปที่ [Google AdSense](https://adsense.google.com/) และเข้าสู่ระบบ
2.  เมนูด้านซ้าย เลือก **Account (บัญชี)** -> **Settings (การตั้งค่า)** -> **Account information (ข้อมูลบัญชี)**
3.  หาบรรทัดที่เขียนว่า **Publisher ID** (รหัสผู้เผยแพร่โฆษณา)
    *   หน้าตาจะเป็นแบบนี้: `pub-1234567890123456`
    *   **สิ่งที่ต้องใช้**: ให้เติม `ca-` นำหน้า เป็น `ca-pub-1234567890123456`

### วิธีแก้ในโค้ด (ไฟล์ `app/layout.tsx`)
1.  เปิดไฟล์ `app/layout.tsx`
2.  เลื่อนลงไปบรรทัดล่างเกือบสุด
3.  หาบรรทัดนี้:
    ```tsx
    src={`https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX`}
    ```
4.  เปลี่ยน `ca-pub-XXXXXXXXXXXXXXXX` เป็นรหัสของคุณ

---

## ขั้นตอนที่ 2: สร้างและรับ Slot ID (รหัสป้ายโฆษณา)
คุณต้องสร้างป้ายโฆษณา 2 ป้าย สำหรับด้านซ้ายและด้านขวา

1.  ใน AdSense เมนูด้านซ้าย เลือก **Ads (โฆษณา)** -> **Overview (ภาพรวม)**
2.  เลือกแถบ **By ad unit (ตามหน่วยโฆษณา)**
3.  เลือกสร้างโฆษณาแบบ **Display ads (โฆษณาแบบดิสเพลย์)**
4.  ตั้งชื่อโฆษณา (เช่น "เว็บเกม-ด้านซ้าย")
    *   **Ad size**: แนะนำเลือก **Fixed** แล้วใส่ขนาด `160x600` (เพื่อให้พอดีกับช่อง) หรือเลือก **Responsive** ก็ได้
5.  กด **Create (สร้าง)**
6.  คุณจะได้หน้าต่างที่มีโค้ด HTML ให้มองหาคำว่า `data-ad-slot="xxxxxxxxxx"`
    *   ตัวเลข `xxxxxxxxxx` คือ **Slot ID** (เช่น `1234567890`)
7.  ทำซ้ำอีกครั้งเพื่อสร้างป้ายด้านขวา (เช่น "เว็บเกม-ด้านขวา", ขนาดแนะนำ `300x250`)

### วิธีแก้ในโค้ด (ไฟล์ `app/page.tsx`)
เปิดไฟล์ `app/page.tsx` แล้วหาจุดแก้ 2 จุด:

**จุดที่ 1: ด้านซ้าย (Left Sidebar)**
```tsx
<AdSenseUnit
   client="ca-pub-XXXXXXXXXXXXXXXX" // <-- ใส่ Client ID เดิม (เหมือนข้อ 1)
   slot="1234567890"               // <-- ใส่ Slot ID ของป้าย "ด้านซ้าย"
   ...
/>
```

**จุดที่ 2: ด้านขวา (Right Sidebar)**
```tsx
<AdSenseUnit
   client="ca-pub-XXXXXXXXXXXXXXXX" // <-- ใส่ Client ID เดิม
   slot="0987654321"               // <-- ใส่ Slot ID ของป้าย "ด้านขวา"
   ...
/>
```

## หมายเหตุ
*   **เมื่อใส่ครบแล้ว**: หน้าเว็บอาจจะโชว์พื้นที่ว่างๆ ขาวๆ ในช่วงแรก (เพราะ Google กำลังตรวจสอบเว็บ) ใช้เวลา 1-2 ชั่วโมง หรือสูงสุด 48 ชั่วโมง โฆษณาถึงจะเริ่มขึ้นครับ
*   **ในเครื่องตัวเอง (Localhost)**: Google มักจะไม่โชว์โฆษณาบน Localhost แต่ถ้าเห็นพื้นที่ว่างๆ ตามขนาดที่กำหนด แปลว่าโค้ดทำงานถูกต้องแล้วครับ!
